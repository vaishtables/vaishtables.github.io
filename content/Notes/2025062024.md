# Student APA 7 w/ Pandoc + Latex

## Checklist
- [ ] Pandoc
- [ ] Zotero

## Cmd for Pandoc

```
pandoc 2507081123.md -o 2507081123.pdf --pdf-engine=xelatex --citeproc --template=apa7.tex
```

### Title Page

- [x] double-spaced
- [x] centered text
- [x] bolded title
- [x] author name format: [First Name] [Middle Initial] [Last Name]
- [x] author affiliation format: [Department Name], [College Name]
- [x] course number & name on same line
- [x] instructor name format: [First Name] [Middle Initial] [Last Name]
- [x] due date format: [Month][D], [YYYY]
- [x] page number flush right in header

\documentclass[12pt, letterpaper]{article}

\captionsetup[table]{justification=justified, singlelinecheck=off}

$if(csl-refs)$
\NewDocumentCommand\citeproctext{}{}

\makeatother
\newlength{\cslhangindent}
\setlength{\cslhangindent}{1.5em}

\setlength{\csllabelwidth}{3em}
\newenvironment{CSLReferences}[2]
{\begin{list}{}{%
    \setlength{\itemindent}{0pt}
    \setlength{\leftmargin}{0pt}
    \setlength{\parsep}{0pt}
    \ifodd #1
    \setlength{\leftmargin}{\cslhangindent}
    \setlength{\itemindent}{-1\cslhangindent}

$endif$

\title{\fontsize{12}{12}\selectfont \textbf{{$title$}}}

\begin{document}
